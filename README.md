# CyFun Automation

Marketing site for CyFun Automation — automated evidence collection and maturity scoring for the
CyFun cybersecurity framework (CCB Belgium), NIS-2 and ISO 27001.

Static site, no build step. Served via GitHub Pages at `cyfun-automation.eu` (see `CNAME`).

Product screenshots in `assets/screenshots/` are from a demo instance seeded with fictional data.
